Vibe Hacks #04 · 3 分钟路演

火锅派对
Hotpot Match

火锅是中国人的社交战场。
我们用 Agent 把它做成 60 秒卡牌策略游戏。

一句话定位

火锅场景 × Agent 角色 × 记忆闪记 × 卡牌策略
—— 1 分钟派对游戏

关键词：一分钟 · Agent 驱动 · 中国社交兵法 · 真的会玩

核心玩法（60 秒一局）

  • 开局 10 秒闪记：记住每种食材烫熟秒数，之后锅里不再显示倒计时
  • 下锅 + 话题炸弹：用政治/排辈/辣度触发 Agent 情绪
  • 拼酒 / 打牌：让对手分心，自己趁机抢菜
  • 3-2-1 抢菜：拼手速，早抢生肉 0 分，完美时机 12 分

技术亮点

Agent 驱动每个角色独立 Profile + LLM 台词
情绪系统话题 → 情绪 → 影响当轮行为
多人同步服务端 RoundState，lobby 选角色同房
记忆设计闪记后隐藏计时，靠脑子判断开抢

演示节奏（3 分钟）

  • 0:00–0:10 开场钩子：「火锅局上抢过肉吗？」
  • 0:10–0:30 闪记环节，强调记忆机制
  • 0:30–1:30 下锅 → 话题 → 拼酒 → 抢菜（高潮）
  • 1:30–2:00 Verdict 结算页
  • 2:00–2:30 技术亮点 30 秒
  • 2:30–3:00 二维码 + 收尾金句
