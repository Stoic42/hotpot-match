# HotpotChat 路演讲稿（故事版）

## 开场钩子（15秒）

> "夏天来了，冬天还会远吗？"
>
> 我是四川人，大学四年在上海，每年9月就开始盼火锅季。但真正的火锅，从来不只是吃——是一场社交兵法。

## 故事引入（30秒）

这游戏里的6个角色，都来自我的大学同学：

- **陈师傅**：重庆室友，火锅原教旨主义者，鄙视一切清汤锅
- **悠微**：政治哲学系二次元，酒精过敏但辣椒无敌
- **Leo**：ADHD 美国交换生，话题跳跃到让你跟不上
- **Marta**：墨西哥学姐，说我们四川辣和墨西哥辣是灵魂亲戚
- **Zion**：戴牙套的美国男生，火锅桌上最谨慎的人
- **Nina**：俄罗斯研究生，话少但能喝

**每个角色背后，都是一个真实的人在火锅桌上的窘迫与高光。**

## 趋势嫁接（30秒）

今年两个东西特别火：
- **Coffee Chat** —— 用一杯咖啡的时间建立弱关系
- **A2A 社交** —— Agent 和 Agent 之间开始自己聊天

但我们中国人不搞 Coffee Chat，我们搞 **Hotpot Chat**。

> **中国 Agent 就要吃中国火锅。**

火锅是强制同步的社交场景：
- 你不能边涮肉边看手机（会被鄙视）
- 你必须记住每种食材烫几秒（否则吃生的）
- 你得会读空气、带节奏、控场

## 核心机制演示（60秒）

### 第一层：记忆挑战
开局10秒，闪现所有食材的烫熟时间。
- 肥牛15秒、毛肚20秒、虾滑8秒...
- 然后倒计时隐藏，全靠记忆

> "这不是考手速，是考你对火锅的熟悉程度——就像真正的火锅局。"

### 第二层：Agent 替身
你不是在扮演角色，你是在训练一个 **Agent 替身**。

- 选悠微？你得学会用政治梗触发她喝酒
- 选陈师傅？用辣度之争让他兴奋
- 每个人有「酒量」和「情绪状态」

**你的 Agent 替身会被激怒、会分心、会喝倒。**

### 第三层：社交操控
- **话题炸弹**：聊政治、论资排辈、二次元入侵
- 让对手陷入情绪 → 该轮无法夹菜
- 你趁机猛吃

### 高潮：抢菜大战
食材熟了，3-2-1倒计时，全屏拼手速。
你的 Agent 替身也会抢——有快有慢，看状态。

## 收尾（25秒）

> "冬天快到了，火锅局要组起来了。
>
> 但这一次，你可以提前训练你的 Agent 替身——
> 让它替你在酒桌上不丢脸，在饭局上吃得饱。
>
> HotpotChat：考验你的 Agent 替身的认知能力，还有酒品。"

**扫码，组一局。**

## 备用互动

如果被问 "为什么是火锅？"

> "咖啡是异步的，可以边喝边看手机。火锅是同步的、强制的、有明确时间窗口的——就像我们正在做的 Agent 实时交互。"
> 
> **中国 Agent 就要吃中国火锅，而不是 Coffee Chat。**

如果被问 "Agent 有什么用？"

> "想象一下，冬天你加班到10点，你的 Agent 替身在火锅局上替你社交、替你抢菜、替你挡酒。你晚到，但它已经打好前站。"

## 技术一句话（如果被问到）
