**一句话定位**: 火锅场景 × Agent 角色 × 记忆闪记 × 卡牌策略的 1 分钟派对游戏

---

## 游戏流程演示（90秒）

### 1. 开局闪记（记忆小游戏）
- 屏幕闪现 10 秒：所有食材的烫熟秒数
- 肥牛 15s / 毛肚 20s / 虾滑 8s / 豆腐 45s...
- **核心设计**: 之后锅里不再显示倒计时，全靠记忆判断何时开抢

### 2. 60 秒局内循环（演示核心操作）
- **下锅**: 点 🔥 选食材，开始计时
- **话题炸弹**: 💬 触发 Agent 反应（聊政治让悠微喝酒，论资排辈让陈师傅怒斥）
- **情绪系统**: Agent 被激怒后该轮无法夹菜
- **拼酒**: ☕ 回合制对战，预制技能 + 轮流出招
- **卡牌**: 🃏 插科打诨（让对手分心 8s）、偷偷下虾滑、敬酒挑战

### 3. 抢菜大战（高潮）
- 食材熟了弹出全屏倒计时 3-2-1
- 拼手速点击"抢！"
- Agent 角色也会参与抢夺，有快/慢/miss 三种结果

### 4. 结算
- 时间到自动跳转 Verdict 页面
- 统计：谁吃得最多、谁酒量最好、谁被激怒次数最多

## 技术亮点（30秒）

| 模块 | 实现 |
|------|------|
| Agent 驱动 | 每个角色有独立 Profile + LLM 调用 |
| 情绪系统 | 话题 → Agent 情绪 → 状态影响（禁夹菜/分心） |
| 同步计时 | 服务端 RoundState，多人同房实时同步 |
| 记忆游戏 | 开局 10s 闪记后隐藏所有倒计时 |
| 无账号设计 | 本地 clientId，扫码即玩 |

## 评委可能问的问题

**Q: 和别的派对游戏有什么区别？**
A: 我们不是单纯拼手速，而是「记忆+策略+社交操控」。你得记住食材秒数，还得读懂每个 Agent 的性格，用话题炸弹让对手分心，自己趁机抢菜。

**Q: Agent 是装饰还是真的影响玩法？**
A: 真的影响。Agent 被激怒后会进入情绪状态（愤怒/分心），该轮无法夹菜。而且每个 Agent 有饮酒力数值，拼酒时直接参与对战。

**Q: 一分钟会不会太短？**
A: 一局 60 秒是紧张感的设计。正好够下 3-4 轮食材，打 1-2 次话题/拼酒，抢 2-3 次菜。快节奏才能多开几局，符合派对场景。

**Q: 商业模式？**
A: 当前黑客松版本验证核心玩法。可扩展方向：1) 角色 DLC（更多地域/职业角色）；2) UGC Agent 市场；3) 品牌联名局（海底捞主题食材/角色）。
